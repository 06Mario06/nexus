Privacy Policy (IT)

Informazioni sul trattamento dei dati per il server Discord Nexus e i bot collegati.

1. Titolare e contatti

  • Titolare: Nexus Team
  • Contatto: nexus.assistancehelp@gmail.com

2. Dati trattati

  • Identificativi tecnici: User ID, Server (Guild) ID, Channel ID, Message ID, timestamp.
  • Eventi di moderazione: azioni, motivazioni, log anti-abuso (se abilitati).
  • I messaggi privati (DM) non sono raccolti intenzionalmente.

3. Finalità

  • Moderazione, sicurezza (anti-spam/anti-raid), prevenzione abusi.
  • Supporto (ticket/segnalazioni) e diagnostica tecnica.

4. Hosting e conservazione log

  • Il sito è pubblicato tramite GitHub Pages (pagine statiche).
  • I log del bot sono salvati su Fly.io per esigenze tecniche e di sicurezza.
  • Regione Fly.io: dati insufficienti per verificare se non configurata/nota. Impostare regione SEE se desiderato.

5. Conservazione (retention)

  • Log moderazione/sicurezza: 90 giorni (modificare se diverso).
  • Ticket/supporto: 180 giorni (modificare se diverso).

6. Diritti

  • Richiesta accesso/cancellazione compatibilmente con esigenze di sicurezza/moderazione.
  • Inviare richiesta via email includendo Discord User ID.

5. Retention

  • Moderation/security logs: 90 days (edit if different).
  • Support/tickets: 180 days (edit if different).

6. User rights

  • Access/deletion requests subject to security/moderation requirements.
