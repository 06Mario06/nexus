Privacy Policy

Nexus • Aggiornato 2025-12-25

Italiano

1. Dati raccolti

Nexus può trattare dati tecnici necessari al funzionamento dei servizi, tra cui: ID utente, ID server, ID messaggi, timestamp e log di moderazione/sicurezza. I messaggi privati (DM) non sono raccolti intenzionalmente.

2. Finalità

I dati sono trattati esclusivamente per: moderazione, sicurezza del server, prevenzione abusi (spam/raid), gestione ticket e applicazione del regolamento.

3. Conservazione

I dati sono conservati per il tempo strettamente necessario alle finalità sopra indicate e possono essere rimossi su richiesta dell’amministratore del server o dell’utente interessato, compatibilmente con esigenze di sicurezza e moderazione.

4. Condivisione

I dati non sono venduti e non sono usati per pubblicità o profilazione. Possono essere trattati da fornitori tecnici necessari.

5. Sicurezza

Vengono adottate misure tecniche e organizzative ragionevoli per proteggere i dati da accessi non autorizzati, perdita o uso improprio.

6. Diritti

Puoi richiedere accesso o cancellazione dei dati (nei limiti delle necessità di moderazione e sicurezza) contattando l’indirizzo email indicato.

7. Hosting

  • Sito: GitHub Pages (contenuti statici)
  • Log tecnici/moderazione: Fly.io

8. Contatto

Per richieste privacy/rimozione dati: nexus.assistancehelp@gmail.com

English

1. Data Collected

Nexus may collect technical data required for its operation, including user IDs, server IDs, message IDs, timestamps, and moderation logs. No private message content is intentionally collected.

2. Purpose of Data Processing

Data is processed exclusively for moderation, server security, abuse prevention, ticket/support handling, and enforcement of server rules.

3. Data Retention

Data is retained only for the time strictly necessary for the service. Data may be deleted upon request by the server administrator or the affected user, subject to security/moderation needs.

4. Data Sharing

Data is not sold, shared, or disclosed for advertising or profiling. It may be processed by necessary technical providers.

5. Data Security

Reasonable technical and organizational measures are implemented to protect data against unauthorized access, loss, or misuse.

6. User Rights

Users may request access to or deletion of their data, subject to moderation and security requirements.

7. Hosting

  • Website: GitHub Pages (static content)
  • Bot technical/moderation logs: Fly.io

8. Contact

For privacy-related inquiries or data removal requests, contact: nexus.assistancehelp@gmail.com
