1. Dati trattati

I bot e gli strumenti di moderazione possono trattare dati tecnici necessari al funzionamento, tra cui: ID utente, ID server, ID messaggi, timestamp e log di moderazione/sicurezza. I messaggi privati (DM) non sono raccolti intenzionalmente.

2. Finalità

  • Moderazione e applicazione del regolamento
  • Sicurezza (anti-spam/anti-raid) e prevenzione abusi
  • Supporto (ticket/segnalazioni) e diagnostica tecnica

3. Conservazione (retention)

I log sono conservati per il tempo necessario alle finalità di sicurezza e moderazione. Imposta qui i valori reali (es. 30/90/180 giorni) se diversi.

4. Hosting e log

  • Sito: GitHub Pages (contenuti statici)
  • Log bot: Fly.io (archiviazione tecnica dei log)

5. Diritti

Puoi richiedere accesso o cancellazione dei dati (nei limiti delle necessità di moderazione e sicurezza) contattando l’email indicata e includendo il tuo Discord User ID.

6. Contatto

Per richieste privacy: nexus.assistancehelp@gmail.com

1. Data processed

Bots and moderation tools may process technical data required for operation, including user IDs, server IDs, message IDs, timestamps, and moderation/security logs. Private messages (DMs) are not intentionally collected.

2. Purposes

  • Moderation and enforcement of server rules
  • Security (anti-spam/anti-raid) and abuse prevention
  • Support (tickets/reports) and technical diagnostics

3. Retention

Logs are kept only for as long as necessary for security and moderation purposes. Adjust retention periods if needed (e.g., 30/90/180 days).

4. Hosting and logs

  • Website: GitHub Pages (static content)
  • Bot logs: Fly.io (technical log storage)

5. User rights

You may request access to or deletion of your data (subject to moderation/security requirements) by contacting the email below and including your Discord User ID.

6. Contact

For privacy requests: nexus.assistancehelp@gmail.com
